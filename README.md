# Portfolio Website

A personal developer portfolio showcasing full-stack projects, including an Influencer Sponsorship Platform (Flask + Vue.js).

## Files
- `index.html` — main static page
- `index.css` — styles
- `index.js` — scripts (if used)

## Live demo
- Influencer platform link inside project card: https://influencer-engagement-sponsorship-sqe4.onrender.com/

## Local setup
1. Clone repository:
   ```bash
   git clone https://github.com/abhibharti5556/Portfolio.git
   cd Portfolio
   ```
2. Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Push code to `main` branch:
   ```bash
   git add .
   git commit -m "Add portfolio"
   git push -u origin main
   ```
2. In GitHub -> Settings -> Pages -> Source -> `main` branch -> `/ (root)`.
3. Use the generated site URL: `https://abhibharti5556.github.io/Portfolio/`

## Notes
- Ensure all files are at repo root.
- For a custom domain, configure in GitHub Pages settings.
